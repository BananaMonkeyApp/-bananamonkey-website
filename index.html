📲 How to Install

BananaMonkey is currently distributed as APK (not on Play Store yet). Follow these steps:

Download the APK

Click "Download App" button at the top. Save the APK file to your phone.

Allow Unknown Sources

Open phone Settings → Apps → Special access → Install unknown apps → Find your browser → Allow

Open the APK File

Tap the downloaded apk file in your Downloads folder or notification.

Tap "Install"

If a warning appears about unknown app, tap "Install Anyway" or "More Info → Install".

Allow Permissions

On first launch, allow Camera, Microphone, Notifications. These are needed for verification & chat.

Sign Up & Start!

Choose Banana (female) or Monkey (male), register, and start your journey!
